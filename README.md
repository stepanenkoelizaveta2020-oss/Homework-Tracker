# Homework-Tracker
Простий трекер для відстеження та виконання домашніх завдань.

# Проблема
Багато школярів і студентів забувають про домашні завдання або згадують про них в останній момент. Через це завдання виконуються поспіхом або взагалі пропускаються. Потрібен простий інструмент, який допоможе зручно фіксувати домашні завдання та контролювати їх виконання

# Пітч
Homework Tracker — це простий додаток для відстеження виконання домашніх завдань. Користувач може додавати нові завдання, бачити їх у списку та відмічати виконаними. Це допомагає краще організувати навчання та не забувати про дедлайни

# Ціль першої ітерації
Створити мінімальну версію додатку, у якій користувач може додавати домашні завдання, бачити список завдань та позначати їх як виконані

# MVP
У мінімальній версії додатку буде реалізований один основний сценарій:
1. користувач додає нове домашнє завдання
2. завдання відображається у списку
3. користувач може позначити його як виконане

# Стейкхолдери
Основними стейкхолдерами є користувачі — школярі та студенти, яким потрібно контролювати свої домашні завдання. Також стейкхолдером є розробник, який створює та підтримує продукт. У рамках навчального проєкту зацікавленою стороною є викладач, який перевіряє результат роботи

# Критерії успіху
1. користувач може легко додати нове домашнє завдання
2. додане завдання відображається у списку
3. користувач може позначити завдання як виконане
4. новий користувач може зрозуміти, як користуватися додатком без пояснень
